Streamgraph

This is a simple streamgraph example that demonstrates the use of the basic components of a streamgraph: area charts, stacking, and smoothing. This is implemented within the Angular.js framework using an Angular directive. This makes it possible to use HTML code like <stream-chart></stream-chart> and have it translated into Javascript code that assembles the data and builds the D3 bar chart.
