dang-streamgraph

This simple streamgraph chart demonstrates how to construct a streamgraph. Streamgraphs are stacked area charts that are adjusted to grow and shrink around the center of the chart, rather than the bottom.

Clicking the transition() button will transition the streamgraph between two data sets.

Made from the command line with vim by charlesreid1.

Made with HTML ( Bootstrap, Bootswatch), Javascript ( D3.js, Angular.js, jQuery), and Python ( Pelican, Seaborn, Matplotlib, Numpy).
